### Start the server:
In both the root folder and the chessus-frontend folder:
`npm run start` or `npm start`

### Create the tables:
First run `npm run start`, then go to `\create-tables`

### Seed the database:
`npm run seed`
or go to `\seed` after running `npm run start`


### Troubleshooting

Connection Refused:
Check to see if the MySQL server is running or not.  If not, go to the root folder and start with `npm run start`.

Can't find module in chessus-node/run: 
Close the terminal, start the server with `npm run start`.
