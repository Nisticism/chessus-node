# Logo and Icon Generation

The GRIDGROVE logo is an SVG file located at `logo.svg`.

## Generating PNG Icons

To generate PNG versions of the logo for different sizes:

1. Open `generate-icons.html` in your web browser (Chrome recommended)
2. Click the download buttons for each size you need:
   - favicon-32x32.png (for favicon)
   - favicon-64x64.png (for favicon)
   - logo192.png (for PWA manifest)
   - logo512.png (for PWA manifest)
3. Save the downloaded files to this directory (`public/`)
4. Optionally, convert favicon PNGs to .ico format using an online tool like favicon.io

## Logo Design

The logo features:
- Three overlapping squares in different rotations representing strategy and board games
- Blue gradient colors (#4a90e2 to #357abd) from the GRIDGROVE brand palette
- A dark navy circle background (#08234d)
- Small strategic dots suggesting game moves
- Subtle rotation and opacity variations creating depth

## Current Icon Files

- `logo.svg` - Vector logo (used in navbar and as primary icon)
- `favicon.ico` - Multi-size favicon for browsers
- `logo192.png` - 192x192 PWA icon
- `logo512.png` - 512x512 PWA icon
